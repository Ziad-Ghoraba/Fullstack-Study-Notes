Func is a **delegate that RETURNS a value** `Func<T1, T2, ..., TResult>`

The **last type is always the return type** and the earlier types are the **input parameters**.

```CS
// One input, one return
Func<int, int> square = (x) => x * x;
int result = square(5); // 25

// No inputs, one return
Func<string> greet = () => "Hello";
```
***
Action is a **delegate that RETURNS void** .. `Action<T1, T2, ...>`

```CS
// One input
Action<string> log = (msg) => Console.WriteLine(msg);

// Two inputs
Action<int, int> printSum = (a, b) => Console.WriteLine(a + b);

// No inputs
Action sayHi = () => Console.WriteLine("Hi!");
```
***
Predicate is a **delegate that always returns a boolean** .. `Predicate<T>`

```CS
Predicate<int> isEven = x => x % 2 == 0;
bool result = isEven(6); // true

// Another example
Predicate<string> longerThan5 = s => s.Length > 5;
```
