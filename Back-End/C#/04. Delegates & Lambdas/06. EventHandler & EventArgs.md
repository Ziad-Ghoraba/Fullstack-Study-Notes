EventHandler is a **predefined delegate** in C# for events.

```CS
void MethodName(object sender, EventArgs e) // Signature
```

- `object sender`:  
    Tells you **who** raised the event (which instance).
- `EventArgs e`:  
    Tells you **what happened** + any **data** you want to send.

By default if you didn't use the angle brackets it will be non-generic so you can't send any data inside the EventArgs.

```CS
public class YouTubeChannel {
	public event EventHandler subscribeEvent;

	public void UploadVideo(string videoTitle) {
		Console.WriteLine($"New Video Uploaded {videoTitle}");
		subscribeEvent(this, EventArgs.Empty);
		// This reference the instance of the class.
		// We used EventArgs.empty because we are using the                  non-generic version .. so we can't send data.
	}
}

public class Subscriber {
	public void Subscribe(YouTubeChannel channel) {
		channel.subscribeEvent += WatchVideo;
	}
	public void WatchVideo(object Sender, EventArgs args) {
		Console.WriteLine("Watch The Video");
	}
}
```
***
Now in the following code we will see an example to pass some value ... and if you want to pass multiple values you can make the EventHandler take a class.

```CS
public class YouTubeChannel {
	public event EventHandler<string> subscribeEvent;

	public void UploadVideo(string videoTitle) {
		Console.WriteLine($"New Video Uploaded {videoTitle}");
		subscribeEvent(this, videoTitle);
	}
}

public class Subscriber {
	public void Subscribe(YouTubeChannel channel) {
		channel.subscribeEvent += WatchVideo;
	}
	
	public void WatchVideo(object Sender, string arg) {
		Console.WriteLine($"Watch The Video {arg}");
	}
}
```

Now we will **Pass a class to hold all the information** we need.

```CS
public class videoInfoEventArgs: EventArgs {
    public string title;
    public int duration;
} // Make it inherit from EventArgs to follow the best practices.

public class YouTubeChannel {
    public event EventHandler<videoInfoEventArgs> subscribeEvent;

    public void UploadVideo(string videoTitle, int timeDuration) {
        Console.WriteLine($"New Video Uploaded {videoTitle}");
        videoInfoEventArgs videoInfo = new videoInfoEventArgs() {
            title = videoTitle,
            duration = timeDuration
        };

        subscribeEvent(this, videoInfo);
    }
}

public class Subscriber {
    public void Subscribe(YouTubeChannel channel) {
        channel.subscribeEvent += WatchVideo;
    }
    
    public void WatchVideo(object S, videoInfoEventArgs eventAr){
	    // We need to cast the object first 
        videoInfoEventArgs videoIn = (videoInfoEventArgs)eventAr;
        
        Console.WriteLine($"Watch The Video {videoIn.title}");
    }
}
```
