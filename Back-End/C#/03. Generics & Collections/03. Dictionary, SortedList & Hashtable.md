**Hashtable** is the same idea as SortedList & Dictionary but it's **non-generic** so it's **not recommended to use it** ... also it's not supporting LINQ.

The difference between Dictionary and SortedList is that **SortedList is sorted based on the key** .. as it uses **Binary Search Tree** to sort them internally.
## Dictionary & SortedList

```CS
Dictionary<string, int> ages = new Dictionary<string, int>();

// 2. Add items
ages.Add("Ali", 25);
ages.Add("Sara", 30);

// 3. Access a value by key
Console.WriteLine(ages["Ali"]);   // Output: 25

// 4. Check if key exists
if (ages.ContainsKey("Ahmed")){
	Console.WriteLine(ages["Ahmed"]);
} else{
	Console.WriteLine("Ahmed not found");
}

// 5. Update a value
ages["Ali"] = 26;

// 6. Remove an item
ages.Remove("Sara");

// 7. Loop through dictionary
foreach (var item in ages){
	Console.WriteLine($"{item.Key}: {item.Value}");
}

// 8. TryGetValue (safe way to get value)
if (ages.TryGetValue("Ali", out int age)){
	Console.WriteLine($"Ali's age is {age}");
}
```
