A **Stack** follows **LIFO** → Last In, First Out.

You push items in a stack and the **last thing you push is the first thing you pop**.

```CS
Stack<int> stack = new Stack<int>();

// Push items
stack.Push(10);
stack.Push(20);
stack.Push(30);

// Peek (top element)
Console.WriteLine(stack.Peek());   // Output: 30

// Pop items (LIFO)
Console.WriteLine(stack.Pop());    // Output: 30
Console.WriteLine(stack.Pop());    // Output: 20

// Check remaining count
Console.WriteLine(stack.Count);    // Output: 1

// Loop through stack
foreach (var item in stack){
	Console.WriteLine(item);       // Output: 10
}
```
***
A **Queue** follows **FIFO** → First In, First Out.

```CS
Queue<string> queue = new Queue<string>();

// Add items (Enqueue)
queue.Enqueue("Ali");
queue.Enqueue("Sara");
queue.Enqueue("Omar");

// Peek (front element)
Console.WriteLine(queue.Peek());   // Output: Ali

// Remove items (FIFO)
Console.WriteLine(queue.Dequeue()); // Output: Ali
Console.WriteLine(queue.Dequeue()); // Output: Sara

// Check remaining count
Console.WriteLine(queue.Count);     // Output: 1

// Loop through queue
foreach (var item in queue){
	Console.WriteLine(item);        // Output: Omar
}
```
