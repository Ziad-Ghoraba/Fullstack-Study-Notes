Collections in C# is a data structure that store classes ... and it can be founded in the namespace `System.Collections`.

It can be divided into **non-generic collections like the ArrayLists** and **generic collections like lists** ... and both are **dynamic** so their size can increase at the runtime.

ArrayLists in C# store objects of any type you can add `int, string, custom object ..etc` because internally it stores everything as **object**.

So ArrayLists take the value from you and do **boxing** as it convert any value you passed it to an object then store it ... when you are reading it then **unboxing** happens. 

```CS
ArrayList list = new ArrayList();
list.Add(10);
list.Add("hello");
list.Add(3.14);
```

**ArrayLists -> old, non-generic, stores anything but not type-safe.**  
**Lists -> modern, generic, recommended.**
***
**List Basic Operations**

```CS
list.Add(5);
list.AddRange(new int[] { 1, 2, 3 }); // Add multiple elements
list.Insert(2, 50); // Insert(index, item)
list.Remove(10); // Remove the first matching element
list.RemoveAt(3); // Remove element at a specific index
list.RemoveRange(0, 3); // RemoveRange(index, count)
bool found = list.Contains(10);
int i = list.IndexOf(5);
list.Clear();
List.Sort(), List.Reverse(), List.Count;
int[] arr = list.ToArray(); // Convert list to array
```
