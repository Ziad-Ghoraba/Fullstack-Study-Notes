**BundlePhobia** is a tool that helps you find the cost of adding a new npm package to our bundle and fine best alternatives.

BundlePhobia benefits:
- Details of the size of a package and it's dependencies.
- Compare the sizes of different versions of a package.
- Download time for the package on different network latencies.
- Tree shaking opportunities and side-effect tree packages.
- Similar packages and optimization opportunities.
- Scan a `package.json` file and give you full analysis for all packages.
