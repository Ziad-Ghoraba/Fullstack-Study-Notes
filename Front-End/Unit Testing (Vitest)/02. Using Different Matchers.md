**toBeTruthy() and toBeFalsy()**

These matchers check whether a value is truthy or falsy in JavaScript, not just `true` or `false`.

Truthy values include: non-empty strings, non-zero numbers, objects, arrays.  
Falsy values include: `false`, `0`, `''`, `null`, `undefined`, `NaN`.

```TS
function getUsername(isLoggedIn: boolean) {
  return isLoggedIn && true;
}

expect(getUsername(true)).toBeTruthy();
expect(getUsername(false)).toBeFalsy();
```

**toBe() and toEqual()**

`toBe()` compares values using strict equality.  
This works well for primitives but fails for objects and arrays because they are compared by reference, not by value.

```TS
function createUser() {
  return { name: 'Hamza' };
}

// primitives → works
expect(5).toBe(5);

// objects → fails (different references)
expect(createUser()).toBe({ name: 'Hamza' }); 
```

`toEqual()` compares objects and arrays by their structure and values, not by reference.

```TS
function createUser() {
  return { name: 'Hamza' };
}

expect(createUser()).toEqual({ name: 'Hamza' }); // Pass
```

**toContain() to check existence in arrays (or strings)**

`toContain()` checks whether a value exists inside an array or string.

```TS
function getRoles() {
  return ['admin', 'editor', 'viewer'];
}

expect(getRoles()).toContain('admin');
```

**.not modifier**

`.not` negates any matcher.  
It can be used with all matchers, not only `toContain`.

```TS
function getRoles() {
  return ['admin', 'editor'];
}

expect(getRoles()).not.toContain('viewer');
expect(5).not.toBe(10);
```

**Testing errors: wrapping functions + toThrow()**

If a function throws an error, you must wrap the function call inside another function.  
If you don’t, the error will be thrown before Vitest can assert it.

```TS
function divide(a: number, b: number) {
  if (b === 0) {
    throw new Error('Division by zero');
  }
  return a / b;
}

// wrong – error thrown immediately
expect(divide(10, 0)).toThrow();

// correct – function wrapped
expect(() => divide(10, 0)).toThrow();
expect(() => divide(10, 0)).toThrow('Division by zero');
```
