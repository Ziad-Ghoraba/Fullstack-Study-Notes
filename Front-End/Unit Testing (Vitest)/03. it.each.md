We can use `it.each` to run the same test logic multiple times with different input data, each row of data becomes a separate test case.

It helps reduce duplication and makes tests easier to read when validating many cases for the same behavior.

```JS
describe('examples.isPrime', () => {
  it.each([
    { num: 0, expected: false },
    { num: 1, expected: false },
    { num: 2, expected: true },
  ])('Should return $expected for $num', ({ num, expected }) => {
    expect(isPrime(num)).toBe(expected);
});
```
