## JavaScript Array Methods

## Array.from

- Converts an iterable or array-like object into an array.
- Syntax: `Array.from(iterable, mapFunc, thisArg)`
    - `iterable`: The source to convert into an array.
    - `mapFunc`: Optional function to apply to each element.
    - `thisArg`: Optional value to use as `this` when executing `mapFunc`.

### Examples

```javascript
// Convert string to array
console.log(Array.from("Osama")); // ["O", "s", "a", "m", "a"]

// Apply map function to each element
console.log(Array.from("12345", (ele) => +ele + +ele)); // [2, 4, 6, 8, 10]

// Create array with unique values from another array using Set
console.log(Array.from(new Set(myArr))); // Array with unique values from myArr
```

## Array.copyWithin

- Copies a part of an array to another location within the same array, overwriting existing elements without changing the array's length.

- Syntax: `Array.copyWithin(target, start, end)`
    - `target`: Index to copy the sequence to. **If equal to or greater than array length, nothing is copied.**
    - `start`: Optional index to start copying from (default: 0). Negative values count from the end.
    - `end`: Optional index to stop copying (not included). Negative values count from the end.

### Examples

```javascript
let myArray = [10, 20, 30, 40, 50, "A", "B"];

// Copy from start to target index 3
myArray.copyWithin(3); // [10, 20, 30, 10, 20, 30, 40]

// Copy from index 6 to target index 4
myArray.copyWithin(4, 6); // [10, 20, 30, 40, "B", "A", "B"]

// Using negative index (same as above)
myArray.copyWithin(4, -1); // [10, 20, 30, 40, "B", "A", "B"]
```

## Array.some

- Tests whether at least one element in the array passes the condition implemented by the callback function. Returns `true` if any element passes, `false` otherwise.

- Syntax: `Array.some(callbackFunc(element, index, array), thisArg)`
    - `callbackFunc`: Function to test each element, returning a boolean.
    - `thisArg`: Optional value to use as `this` when executing `callbackFunc`.

### Use Cases

1. Check if an element exists in the array.
2. Check if a number is within a specific range.

### Examples

```javascript
let nums = [1, 2, 3, 4];

// Check if any element is greater than 4
console.log(nums.some((ele) => ele > 4)); // false

// Using thisArg
let testNum = 3;
let result = nums.some(function (ele) {
  return ele > this;
}, testNum); 
// true (checks if any element is greater than testNum)
```

## Array.every

- Tests whether all elements in the array pass the condition implemented by the callback function. Returns `true` if all elements pass, `false` if any fail.

- Syntax: `Array.every(callbackFunc(element, index, array), thisArg)`
    - `callbackFunc`: Function to test each element, returning a boolean.
    - `thisArg`: Optional value to use as `this` when executing `callbackFunc`.

## Spread Operator

- Allows iterables (like arrays or strings) to expand in place.
- Syntax: `...iterable`

### Use Cases and Examples

1. **Spread with String**:

```javascript
console.log(..."Osama"); // "O", "s", "a", "m", "a"
console.log([..."Osama"]); // ["O", "s", "a", "m", "a"]
```

2. **Concatenate Arrays**:

```javascript
let finArr = [...myArr1, ...myArr2]; 
// Combines myArr1 and myArr2 into a new array
```

3. **Push Inside Array**:

```javascript
allFriends.push(...otherFriends); 
// Adds elements of otherFriends to allFriends
```

4. **Use with Math Object**:

```javascript
console.log(Math.max(...nums)); // Finds max value in nums array
```

5. **Merge Objects**:

```javascript
console.log({ ...objOne, ...objTwo, e: 5 }); 
// Merges objOne and objTwo, adds property e
```
