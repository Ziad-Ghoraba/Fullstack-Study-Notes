
## Control Flow

- **if**, **else if**, **else**
- Ternary operator: `condition ? true : false`
- **switch**: Uses `===`, requires `break`

```js
let result = theGender === "Male" ? "Mr" : "Mrs";
console.log(`Hello ${theGender === "male" ? "mr" : "mrs"} ${theName}`);
```

## Arrays

- Create: `new Array()` or `[]`
- Access: `array[index]`
- Check: `Array.isArray()`
- Modify length: `array.length = n`

```js
let myArray = ["hamza", "ehab", "mahrous"];
myArray[myArray.length] = "newElement"; // Add dynamically
```

### Array Methods

- **Add/Remove**: `push()`, `pop()`, `unshift()`, `shift()`, `splice(start, deleteCount, ...items)` ... you can use `push` and `unshift` to add more than one element ... and note that the methods that removes an element it returns it.

- **Search**: `indexOf()`, `lastIndexOf()`, `includes()`
- **Slice**: `slice(start, end)` (new array, no modify)

- **Sort**: `sort()` depends on the alphabetical order .. and it modifies the original array ... `reverse()` (modifies original also).

- **Join**: `concat()`, `join(separator)`

```js
myArray.splice(3, 1, "hamza", "mahrous"); // Modify original by starting from index 3 then remove one item then add the rest.

let result = arr1.concat(arr2, ["hamza", "ehab"]);
console.log(fruits.join(" - ")); // "Apple - Banana - Cherry"
```

## Loops

- **for**, **while**, **do...while**
- Loop control: `break`, `continue`, `label`

```js
for (;;) { // Infinite loop, keep semicolons
}
while (condition) {
  // Block
}
do {
  // Block
} while (condition); // At least excuted once.
```
