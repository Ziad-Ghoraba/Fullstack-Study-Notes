## Window Object

- Represents the browser window
- Contains the Document Object Model (DOM)
- Capabilities:
    - Open, move, close, or resize the window
    - Print document or parts of it
    - Set wait time for code execution
    - Full control over the URL
    - Save data

### Dialog Methods (General Knowledge)

- `alert(Message)`: Displays a message with only an "OK" option
- `confirm(Message)`: Displays a message, requires response, returns a Boolean
- `prompt(Message, Default Message)`: Collects user input

### Timeout and Interval Methods

- `setTimeout(Function, Timeout, Additional Params)`:
    - Executes a function after a specified delay
    - Additional parameters are passed to the function
    - Returns a `timeoutID` (positive integer identifier)

- `clearTimeout(Identifier)`: Cancels a timeout to prevent function execution


- `setInterval(Function, Milliseconds, Additional Params)`:
    - Repeatedly calls a function at specified intervals
    - Same parameters as `setTimeout`
    
- `clearInterval(Identifier)`: Stops the interval execution

## Location Object

- Part of the BOM, a property of the `window` object
- Contains information about the current URL
- Provides methods for redirecting the browser (triggers reload/navigation)

### Properties and Methods

- `href`: Gets/sets the entire URL (can redirect to websites, page sections, files, or mail)
- `host`: Returns hostname and port
- `hostname`: Returns only the hostname
- `hash`: Returns the anchor part (links to specific page sections) .. `#about`
- `protocol`: Returns the protocol (`http` or `https`)
- `reload`: Reloads the current page
- `replace`: Replaces the current page with a new one (removes from history)
- `assign`: Loads a new page (keeps history entry)

### Key Concepts

- **Hostname**: Domain name of the server (human-friendly alternative to IP addresses)
- **Port**: Communication endpoint
- **Protocol**: Defines data transfer method
    - `http`: Unsecured
    - `https`: Secured with SSL/TLS encryption

## Window Management

- `open(URL [opt], Window Name or Target Attr [opt], Win Features [opt], History Replace [opt])`:
	- Opens a new window
	- Window Features:
		- `left`
		- `top`
		- `width`
		- `height`
		- `menubar` (`yes` or `no`)

- `close()`: Closes windows opened by the `open` method

## History Object

### Properties

- `length`: Number of entries in the history stack

### Methods

- `back()`: Navigates one step back in history
- `forward()`: Navigates one step forward
- `go(Delta)`:
    - `-1`: One step back
    - `0`: Reloads the page
    - `1`: One step forward
    
- `pushState` and `replaceState`: Manipulate history **stack** without page refresh

## Scrolling

- `scrollTo(x, y | options)`:
    - Scrolls to specific coordinates
    - Example:

```javascript
window.scrollTo({
	left: 2000,
	top: 3000,
	behavior: "smooth"
});
```

- `scroll(x, y | options)`: Similar to `scrollTo` (browser support differs)
- `scrollBy(x, y | options)`: Scrolls by the specified amount
- `scrollX`: Read-only, returns horizontal scroll position
- `scrollY`: Read-only, returns vertical scroll position
- `stop()`: Stops page content loading
- `print()`: Prints the page
- `focus()`: Sets focus on a specific window

## Local Storage

- Persistent storage with no expiration
- HTTP and HTTPS data are isolated
- Private/incognito mode isolates storage from normal browsing

### Methods

- `setItem(key, value)`:
    - Example: `window.localStorage.setItem("color", "#F000")`
    - Alternatives: `window.localStorage.color = "#F000"` or `window.localStorage["color"] = "#F000"`

- `getItem(key)`: Retrieves value by key
    - Example: `window.localStorage.getItem("color")`

- `removeItem(key)`: Removes a specific key-value pair
    - Example: `window.localStorage.removeItem("color")`

- `clear()`: Clears all local storage
- `key(n)`: Returns the name of the nth key

## Session Storage

- Same methods as Local Storage
- Session-specific behavior:
    - New tab = new session
    - New tab with same URL = new session
    - Duplicate tab = copied session
