## Strings

- Use escape operator `\` for special characters or line continuation
- Concatenation with `+` or `console.log(a, b, c)`
- Template literals with backticks `` ` `` and `${variable}`

```js
console.log("elzero\nweb\nschool"); // Newline
console.log(`Hello ${name}`); // Template literal
```

### String Methods

- `charAt()`, `length`, `trim()`, `toUpperCase()`, `toLowerCase()`
- `indexOf(value, start)`, `lastIndexOf(value, start)`: Returns -1 if not found.

- `slice(start, end)` .. not includes the end .. if the end isn't specified will slice to the end and also can start from a negative number, `repeat(n)`, `split(separator, limit)` .. the split convert the string to an array depending on the separator .. The limit is used to define the number of items you want.

- `substring(start, end)` .. (start > end .. will swap) and (tart < 0 .. it will start from zero) .... `substr(start, length)` 

- `includes(value, start)`, `startsWith(value, start)`, `endsWith(value, length)`

```js
console.log(theName.trim().charAt(2).toUpperCase());
console.log(theName.slice(0, 5)); // Slices string
console.log("#".repeat(20)); // Repeats string
```
