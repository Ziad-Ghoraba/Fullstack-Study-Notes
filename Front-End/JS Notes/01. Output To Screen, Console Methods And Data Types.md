## Output to Screen

```js
window.alert("warning"); // Not recommended
document.write("<h1>span</h1>"); // Writes HTML or text to document
console.log("hello world"); // Logs to console
```

### Console Methods

- `log`: Prints to console
- `error`: Displays error in console
- `table`: Prints array in tabular form
- `group`: Creates console group (use `console.groupEnd()` to close)

```js
console.error("error");
console.table(["ahmed", "hamza"]);
console.log("This is my %ctraining", "color:red; font-size:30px");
```

## Data Types

- **String**: `console.log("hamza")`
- **Number**: `console.log(5000)` or `console.log(5.1)` (integer/float same)
- **Array**: `console.log([10, 14, 16])` or `console.log(["as", "ha", "ma"])`
    - Check with `Array.isArray()`

- **Object**: `console.log({ name: "ha", age: 20 })`
- **Boolean**: `console.log(typeof true)`
- **Undefined**: `console.log(typeof undefined)` ... Undefined is used in JS to to indicate a variable that has been declared but not assigned a value.
- **Null**

let myUser = {
