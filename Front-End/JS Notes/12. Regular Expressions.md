## Overview

- **Definition**: Regular expressions (regex) are sequences of characters forming a search pattern, used with string methods to find, replace, or extract information from strings.

- **Use Cases** ... 
- **Validation**: Validate user input (e.g., email addresses, phone numbers).
- **Searching**: Find specific patterns in a string (e.g., words starting with "a").
- **Replacement**: Replace patterns in a string (e.g., replace "colour" with "color").
- **Splitting**: Split a string into an array of substrings (e.g., split text into sentences).

## Syntax

- **Pattern Format**: `/pattern/modifier(s);`
    - Wrap patterns between forward slashes (`/`).
    - Example: `/elzero/` matches the exact sequence "elzero" in a string.
    
- **Alternative Syntax**: `new RegExp("pattern", "modifier(s)")`

## Modifiers (Flags)

- **i**: Case-insensitive (e.g., `/elzero/i` matches "Elzero", "ELZERO", etc.).
- **g**: Global (finds all matches, not just the first).
- **m**: Multiline (makes `^` and `$` match the start/end of each line in a multiline string).

## Search Methods

- **match(pattern)**:
- Used on a string, takes a regex pattern as an argument.
- Returns an array of matches or `null` if no match is found.

```js
let myString = "Hello Elzero Web School I Love elzero";
let regex = /elzero/gi;
console.log(myString.match(regex)); // ["Elzero", "elzero"]
```

## Ranges

- **Character Sets**: Defined with square brackets `[]`.
    - `[0-9]`: Matches any digit from 0 to 9.
    - `[^0-9]`: Matches any character not a digit.
    - `[a-z]`: Matches any lowercase letter.
    - `[^a-z]`: Matches any character not a lowercase letter.
    - `[A-Z]`: Matches any uppercase letter.
    - `[abc]`: Matches any single character: "a", "b", or "c".
    - `[^abc]`: Matches any character not "a", "b", or "c".
    - Example: `/Os[5-9]Os/` matches "Os5Os", "Os6Os", etc.

- **OR Operator**: `(X|Y)` matches either X or Y (e.g., `/(info|org|io)/gi`).

## Character Classes

- **`.`**: Matches any character except newline or line terminators.
- **`\w`**: Matches word characters (`[a-zA-Z0-9_]`).
- **`\W`**: Matches non-word characters.
- **`\d`**: Matches digits (`[0-9]`).
- **`\D`**: Matches non-digits.
- **`\s`**: Matches whitespace characters.
- **`\S`**: Matches non-whitespace characters.
- Example: `/\d\d:\d\d/` matches time formats like "12:34".
- Example: `/\w@\w\.(com|net)/g` matches emails like `"[user@domain.com](mailto:user@domain.com)"`.

## Character Classes Part 2 & Test Method

- **`\b`**: Matches the beginning or end of a word.
    - Example: `/\bcat\b/` matches "cat" but not "category" or "concatenate".
- **`\B`**: Matches positions not at the beginning or end of a word.
- **Test Method**: `pattern.test(input)`
    - Returns `true` or `false` based on whether the pattern matches.
    - Example:

```js
console.log(/(\bspam|spam\b)/gi.test("spam1")); // true
```

## Quantifiers

- **n+**: Matches one or more occurrences.
- **n***: Matches zero or more occurrences.
- **n?**: Matches zero or one occurrence.
- **{n}**: Matches exactly `n` occurrences (e.g., `/a{3}/` matches "`aaa`").
- **{n,}**: Matches `n` or more occurrences.
- **{n,m}**: Matches between `n` and `m` occurrences (inclusive).

```js
let serials = "S123S S1234S S12345S";
console.log(serials.match(/S\d{4,5}S/g)); // ["S1234S", "S12345S"]
```

```js
let urls = "https://google.com http://www.website.net web.com";
let urlsRe = /(https?:\/\/)?(www.)?\w+.\w+/gi;

console.log(urls.match(urlsRe));
// ["https://google.com", "http://www.website.net", "web.com"]
```

## Quantifiers Part 2

- **$**: Asserts the end of a line (e.g., `/ing$/` matches strings ending with "ing").
- **^**: Asserts the start of a line (e.g., `/^we/` matches strings starting with "we").
- **(?=...)**: Positive lookahead (matches if followed by something).
- Example: `/\d\w{5}(?=Z)/gi` matches a digit followed by 5 word characters followed by "Z".
- **(?!...)**: Negative lookahead (matches if not followed by something).
- Example: `/\d\w{8}(?!Z)/gi` matches a digit followed by 8 word characters not followed by "Z".

## Replace Methods

- **replace()**: Replaces the first occurrence of a pattern.
- **replaceAll()**: Replaces all occurrences.
- Using regex:

```js
let text = "Hello @ World @";
console.log(text.replace(/@/gi, "JavaScript")); 
// "Hello JavaScript World JavaScript"
```    

## Input Form Validation

- Example: Phone number validation
```js
let phoneRegex = /\(\d{4}\)\s\d{3}-\d{4}/gi; 
// Matches (1234) 567-8910
```    

- Note: Special characters like `()` must be escaped with `\`.

## Additional Notes

- **IPv6 Address**: Represented as eight groups of four hexadecimal digits, separated by colons.
- Example: `/^[0-9a-f]{1,4}(:[0-9a-f]{1,4}){7}$/i`
- **Non-Capturing Group**: `(?:...)` groups without capturing for output.

```js
console.log("ac".match(/(a|b)c/)); // ["ac", "a"]
console.log("ac".match(/(?:a|b)c/)); // ["ac"]
```

- **Data Format Manipulation**:

```js
let date = "12-31-2020";
let transformedDate = date.replace(/(\d{2})-(\d{2})-(\d{4})/, "$3-$1-$2"); 
console.log(transformedDate); // "2020-12-31"
```
