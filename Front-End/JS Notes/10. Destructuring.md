## Array Destructuring

### Basic Array Destructuring

- Extracts elements from arrays into distinct variables.
- Example:

```javascript
let myFriends = ["ahmed", "osama"];
let [a, b] = myFriends;

console.log(a); // Outputs: "ahmed"
```


### Skipping Items

- Use commas to skip elements during destructuring.
- Example:

```javascript
myFriends = ["ahmed", "osama", "mohamed"];
[a, , c] = myFriends;
console.log(a); // Outputs: "ahmed"
console.log(c); // Outputs: "mohamed"
```


### Rest Operator

- Collects remaining elements into a new array using `...`
- Example:

```javascript
myFriends = ["ahmed", "osama", "mohamed", "ali"];
[a, ...others] = myFriends;

console.log(a); // Outputs: "ahmed"
console.log(others); // Outputs: ["osama", "mohamed", "ali"]
```    

### Default Values

- Assigns default values to variables if the array element is undefined.
- Example:

```javascript
myFriends = ["ahmed"];
[a, b = "default"] = myFriends;
console.log(a); // Outputs: "ahmed"
console.log(b); // Outputs: "default"
```


### Nested Arrays

- Destructures elements from nested arrays.
- Example:

```javascript
let nestedArray = ["ahmed", ["osama", "mohamed"]];
let [a, [b, c]] = nestedArray;

console.log(a); // Outputs: "ahmed"
console.log(b); // Outputs: "osama"
console.log(c); // Outputs: "mohamed"
```    

### Swapping Variables

- Swaps values of variables using destructuring.
- Example:

```javascript
let book = "video";
let video = "book";

[video, book] = [book, video];
```

## Object Destructuring

### Basics

- Extracts properties from objects into variables using property names.
- Must use the exact property names from the object.
- Use parentheses `()` if variables are already declared.
- Example:

```javascript
let user = { 
	theName: 'John',
	theAge: 30, 
	theTitle: 'Mr.', 
	theCountry: 'USA' 
};

({ theName, theAge, theTitle, theCountry } = user);
```    

### Renaming Variables

- Renames properties during destructuring using `property: newName` syntax.
- Incorrect renaming results in `undefined`.
- Example:

```javascript
let person = { name: 'John', age: 30 };
let { name: firstName, age: years } = person;

// Incorrect
let { firstName, years } = person;
console.log(firstName); // Outputs: undefined
```


## Mixed Content Destructuring

- Combines object and array destructuring for complex data structures.
- Example:

```javascript
const data = {
  name: 'John',
  age: 30,
  address: {
	city: 'New York',
	country: 'USA'
  },
  hobbies: ['Reading', 'Swimming', 'Programming'],
};

let {
  name,
  age,
  address: { city, country },
  hobbies: [hobby1, , hobby3],
} = data;
```


## Notes

- **Arrays**: Cannot destructure and rename in the same line; requires two steps.
- **Objects**: Allows skipping properties without placeholders, unlike arrays.
